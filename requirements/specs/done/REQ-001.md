## Summary

`widgets/auth/ui/auth-section.tsx`를 PascalCase + named export로 정리하고, `pages/riot/index.tsx`가 슬라이스 barrel 경유로 import하도록 한다.

## Background

`widgets/auth/index.ts`에 `import AuthPage from "./ui/AuthSection"`이 있는데 실제 파일은 `auth-section.tsx`다. macOS는 case-insensitive라 동작하지만 Linux 빌드(Docker)에서 깨질 수 있다. 같은 김에 default export · deep import도 정통 컨벤션으로 정리.

## Requirements

- `auth-section.tsx` → `AuthSection.tsx` rename
- default export → `export const AuthSection`
- `widgets/auth/index.ts`에서 `AuthSection` named re-export
- `pages/riot/index.tsx`가 `@/widgets/auth` barrel 경유로 import
